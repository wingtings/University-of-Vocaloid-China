# 8th.「三月雨」

**P主：Days的Wing翼**

> 《生查子·三月雨》
>
> 往日歌楼头，你我曾重游。且听君琴悠，且舞我云袖。
> 今日穿时空，唯吾独自留。无缘再聚首，不尽相思愁。

三月微风，烟雨江南，佳人苦苦等候，将军何时还乡？绵绵春雨到无期，漾起心湖水中影。

涉江采芙蓉，将以遗所思。杨柳岸，晓风残月，英俊的将军抚琴低吟，动人的佳人翩翩起舞。良辰美景易逝，到如今，重唱此曲却已无你。

曲中的经典词句各位想必也是耳熟能详，它在Vsinger Live 2017、24年的无限共鸣演唱会均有出现。

![另外一张三月雨.png](../CN_fig/%E5%8F%A6%E5%A4%96%E4%B8%80%E5%BC%A0%E4%B8%89%E6%9C%88%E9%9B%A8.png)

在遥远的的2012，洛天依v3声库刚刚发售，中文vocaloid刚刚破土而生。这是一个早期经典的“阿绫领盒饭天依守寡”剧情，因为那时乐正绫的人设虽然已经完善，但是一直到15年才发布了v3声库，“三月雨下成了三年雨”。三年的时间里就只能以这种形式写歌，也是一种时代特色罢。
虽然声库性能有限，但那个时代的创作者各显神通，用自己的细心打磨弥补技术的沟壑，让带着略微电音的声音形成独特的风格。13年过去，歌声合成技术越来越先进，声音越来越真实，记忆中的初始之音却大抵是越来越模糊了罢。
你问我三月雨怎么四月才发，那原因当然是，这首经历了13年日月沉浮的歌曲，在此时此刻终于达成了一百万播放。彼时千丝万缕的三月雨，给世人留下了传说。
