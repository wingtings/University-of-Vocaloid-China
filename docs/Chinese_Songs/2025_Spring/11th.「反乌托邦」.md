# 11th.「反乌托邦」

**投稿日期：2025/2/3**

**P主：Utopia_乌托邦P**

![反乌托邦.jpg](../CN_fig/%E5%8F%8D%E4%B9%8C%E6%89%98%E9%82%A6.jpg)

> 似乎很有趣的是，名为反乌托邦的歌曲，却是由名为乌托邦之人谱成。
> 
> ——题记

是什么魔力，让一个去年12月才初次投稿的新人写出的第三首原创摇滚曲，在短短几天登上B站热门，入选每周必看，周刊霸榜，到现在应经有了五百五十多万的惊人播放量？

不同的人会给出不同的答案，对于笔者而言，第一次听到后的热泪盈眶便是最好的答案，全是感情，没有技巧。
想起几个月前，笔者在背井离乡上大学之前，留给学弟学妹们的话便是：生活并不像你想象得那么好，也不像你想象得那么坏。这个世界必然容不下所有人的理想，所有人的愿望，所以世界本是反乌托邦，而向往乌托邦之人，必先认清这个世界的荒谬，才能坚定不移地守护自己的正义。

一个偶然的机会，听闻在复旦的挚友遭遇了一些打击，笔者当即把这首歌分享给他。令人欣慰的是，星尘和诗岸动人的歌声抚慰了他的心灵。他听了这首歌后感慨万千，忧郁痛苦一扫而空。以下内容来自他沉吟许久得到的感想：

有趣的，留在记忆深处的似乎总是些虚无飘渺的东西。人站在不同的时刻与视角看待一件事确实会感受不同啊，但可能我刚刚还满于在现世的欢愉中，可当一听到这首歌，回忆涌上心头，确实止不住的想要写下一些浅薄的文字了，也许总是些刻骨铭心的。

“两块五的签字笔，东拼西凑的词句，妄想用它拯救你，顺便拯救我自己。”我和朋友打趣似的说过，任何一个经历过中式教育的人看到这句话多多少少会有些共鸣的，那支再寻常不过的签字笔，却又承载了多大的重量：希望？勇气？力量？当你从书海里突然惊醒时，那支沙沙的笔尖却又一刻不停歇的动着，你凝视着它，是陌生还是熟悉？书写下的文字在这一刻是实实切切的，可它背后投射出来的未来却似乎虚幻，在虚无残梦中和着这样的歌曲，你思绪又猛地回到了现世…它真的能拯救你吗？似乎不能，可是你那渴望自由的心脏，你那滚烫发热的灵魂，它们驱使着，一定要做些什么，你偏要去见证海的那边是否是真的自由，你偏要在这斗室里铺满绝句。你知道没有真正的乌托邦，世界是一个巨大的反乌托邦，但你，手握刀剑，仍然蹀躞在追寻乌托邦的路上。

反乌托邦中的争夺 把愿望敲碎成粉末 心灵也被挂上门锁 看不清灯火 但我还想做些什么 其实还想要说很多 请你等等我 请你等等我 两块五的签字笔 东拼西凑的词句 妄想用它拯救你 顺便拯救我自己 只因为 你那渴望自由的心脏 困在一张没空隙的网 我们的周围并非没光亮 只是太耀眼将我们灼伤 在这个世界难免会迷茫 但别再把小刀带在身上 至少我还在为你而歌唱 在黑暗漫长的反乌托邦 即便我们都在这黑暗漫长的反乌托邦忍着想要挣脱的生活 做出不想做的承诺 列车上坐错站的乘客 困惑着沉默 但我还想做些什么 其实还想要说很多 请你等等我 请你等等我 发着光的半平米 胡乱哼唱的旋律 但是还想拯救你 同时拯救我自己 就请你 放心大胆更妄为肆意 迈步向前别害怕失去 没有犯下任何错的你 在我心里只用做自己 这段胡乱拼凑的词语 如果伤到你也请你忘记 但若你又找不到意义 希望这首歌能被记起 活下去 活下去 活下去 活下去 在这反乌托邦里

世界本是反乌托邦，但是对乌托邦心怀追寻和向往的人们不应被奚落和嘲笑啊。

无论多么令人失望的结果，也请一直活下去啊，在这反乌托邦里。
