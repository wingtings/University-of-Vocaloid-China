# 2nd.「Summering」
**P主：じん**
**「Summering」——盛夏终将止息，但挚爱与回忆永不毁灭。**
去年十月份，じん一首Summering，把我们重新带回那个属于阳炎的夏日。
昔日一去不返，背负着过往的影，我们回转向前。
在执笔与绝笔的踌躇中，失去一切的觉悟凝铸。
