# Welcome to VSTC!

大家好！这里是中国科学技术大学地球和空间科学学院学生虚拟歌手创作文化和歌声合成技术小组（名字这么长，其他人做得到吗.jpg），简称**歌声合成技术协会**，或者**歌研社**。

USTC歌声合成技术协会，是科大爱好歌声合成创作与文化的同好者的大本营。我们期待着所有对歌声合成感兴趣的同学的加入！
我们每个人内心深处都埋藏着丰富而细腻的情感，而歌声正是传递心声、激发共鸣的最好载体。歌声合成技术出现之前，非专业音乐人想要唱出承载自己心绪的歌曲，若请一位歌手，成本是难以想象的。自从歌声合成技术出现，配合电子作曲软件，我们便可能以较低的成本将自己独特的情怀寄托在歌声之中。虚拟歌手自诞生之时起，就承载着让万万千千年轻人的心声被尊重，被表达，被理解的使命。我们用创作，用热爱，用真情使二次元的歌手们鲜活立体。“她把歌声借给我一半，我把灵魂送给她一半”。
在这个温馨的大家庭里，你不仅可以与技术高超的producer交流调校经验、在观影会上尽情欣赏优秀的歌声合成作品，更可以运用歌声合成技术创造出属于自己的歌声，表达自己独一无二的情感。相信你一定能在这里摸索出自己心的音律！
感兴趣的同学们可以多多关注！

## 关于我们

![来中术大学学中术技术.png](img/%E6%9D%A5%E4%B8%AD%E6%9C%AF%E5%A4%A7%E5%AD%A6%E5%AD%A6%E4%B8%AD%E6%9C%AF%E6%8A%80%E6%9C%AF.png)

本社团发源于QQ群聊[USTC中V学园](https://qm.qq.com/q/rQDnmJWE3A)，该群建于2024年7月23日，致力于成为科大中术人的聚集地。在这里，你不仅可以找到同好，和同好一起水群，而且可以发挥自己的特长，唱、弹、调、画、写、绘等等，与群友一起完成让人叹为观止的作品，也可以参加团建（唱K、出cos、看演唱会等应有尽有！），与群友度过愉快难忘的时光。希望有一天，我们能一起写下“洛天依原创”，在科大留下耀眼的痕迹。

24年的游园会期间，我们初步讨论了建立社团的可能性，开了若干会议进行讨论，组建了最初的干事团队，在高年级学长的帮助下于10月份建立了院级社团，挂靠在地空学院团委名下。一年以来，我们尝试了多种活动形式，积累了一定组织活动的经验，这些宝贵的回忆将呈现在各个页面中，供各位观赏。

如果需要编辑器、声库、数字专辑等资源，请加入USTC中V学园的[科大云盘群组](https://pan.ustc.edu.cn/group/home/60000028)，这里收集了丰富的学习资源，可以减轻您寻找各类有用资源的痛苦。



## 欢迎你们

如今我们也在考虑招揽新鲜血液的加入，于是组建了[歌声合成协会干事过渡群](https://qm.qq.com/q/HV5AzqZZuK)，如果你想参与到社团的具体运营、提出建议、寻求帮助与合作，欢迎加入！（ldx们需要接班人们的大手）

你可以在这个群里投稿自己创作的推歌稿件（QQ官号每周推送，一周日V一周中V），也可以参与策划社团的原创曲项目（很需要有能力的人来帮忙！），亦或讨论歌声合成技术甚至参与调校的周常活动准备，绘画大手子还能参加[PV分部](https://qm.qq.com/q/hPJ3be3oAw)，不定期向群友分享自己的摸鱼产物，为P主提供PV素材。
